# 站点战略

## 定位

中文免费在线工具箱，首批工具：JSON、Base64、MD5、图片处理、Clash 节点合并。

## 域名

- 主站：`tools.cqzzz.top`
- 品牌：轻工具箱

## 阶段

1. **冷启动（当前）**：核心工具上线 + 基础 SEO + 提交 GSC
2. **增长期**：按数据优化 title/内链，每周 0-1 个新工具
3. **扩展期**：按数据增加流量模块（资讯聚合、资源页等），仍坚持质量门槛
4. **变现期**：日 UV 稳定后申请 AdSense

## 语言策略

- **主语言：中文**（你的优势 + 百度/搜狗/GSC 中文查询）
- 工具页 URL 保持英文 slug（json-viewer）利于分享与开发习惯
- 后期可选：热门工具加英文版页面做第二流量层，非必须

## 不做

- 医疗/法律/金融建议
- 纯 AI 洗稿文章站
- 采集站、站群
